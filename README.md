# Automatidata Taxi Data Analysis

**Python • pandas • Data Inspection • Exploratory Data Analysis (EDA) • PACE Framework**

## Overview

This project demonstrates the early stages of a data analytics workflow using Python and pandas. Working within the fictional Automatidata consulting scenario, the objective was to inspect, organize, and summarize New York City Taxi & Limousine Commission (TLC) trip data before exploratory data analysis and predictive modeling.

The project focuses on understanding the dataset, identifying relevant variables, detecting potential data quality issues, and preparing the data for future analysis.

---

## Scenario

Automatidata has been contracted by the **New York City Taxi & Limousine Commission (TLC)** to help develop a regression model capable of predicting taxi fares.

Before modeling can begin, the dataset must be inspected to:

- Understand the available variables
- Verify data types
- Identify missing values and anomalies
- Generate descriptive statistics
- Prepare the dataset for exploratory data analysis (EDA)

This project represents the first stage of that analytical workflow.

## Key Tasks Completed

- Imported and loaded the NYC Taxi dataset
- Inspected dataframe structure using `head()`, `info()`, and `describe()`
- Reviewed column data types and non-null values
- Sorted and analyzed trip distance and total fare values
- Investigated payment methods and vendor information
- Filtered datasets using Boolean indexing
- Calculated grouped summary statistics using `groupby()`
- Prepared data for future exploratory analysis and predictive modeling
